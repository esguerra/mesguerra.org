Parameters

In 3DNA there are 3 sets of ``symmetric'' local parameters being defined which are:

  1. Base-pair parameters.
  2. Base (Base pair) step parameters.
  3. Base (Base pair) helical parameters.

Bases or base-pairs are treated as ``rigid bodies'' using classical mechanics.

The first two sets of parameters are based on cartesian coordinates, whereas the third is referred as helical coordinates, which resembles cylindrical coordinates and it's based on Chasles Theorem[1].
