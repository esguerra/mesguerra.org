Ortep
Ortep is a molecule drawing program. It works with cartesian coordinates and it can draw bonds with distance constraints supplied by the user.
It is not an friendly user interface that of Ortep but it gets the job done nicely. You can download statically linked binaries for Linux from:
Once you have unpacked and have added the $PG_PLOT environment variable to your .bash_profile then you can start using the program.
You need to make and input file, I have and example for benzene, you can give it any name an extension. The example is benzene.ortep
