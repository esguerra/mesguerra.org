SVN Cheatsheet


To checkout (i.e. copy) a repository. 
svn co https://dnagit.rutgers.edu/scm/Papers/RNAMotifs
To check the status of the project. 
svn status
To add files or folders to the project. 
svn add *
svn add file
svn Folder/
To commit changes to the project. 
svn commit
To remove files. 
svn rm file
svn rm folder
